`v0.1.0` is the first tagged Querylane preview.

- Established the Go and React application foundation for a PostgreSQL admin UI.
- Added background PostgreSQL metric sampling probes.
- Added multi-architecture container publishing with stable and development channels.
- Established frontend release tagging and the initial continuous-integration path.

This tag is a development preview, not a compatibility or support commitment. Read [Unreleased](/changelog/unreleased) for the substantial product work added since the tag.
