## ExecuteQuery

Executes a single read-only SQL query and streams results in batches.

```protobuf
rpc ExecuteQuery(ExecuteQueryRequest) returns (stream ExecuteQueryResponse)
```

- **Procedure:** `POST /querylane.console.v1alpha1.SQLService/ExecuteQuery`
- **Mode:** Server stream

### JSON request example

The values are illustrative. Replace resource names, identifiers, and credentials for your deployment.

```json
{
  "parent": "instances/production/databases/app",
  "statement": "SELECT current_database();",
  "rowLimit": 100,
  "batchSize": 500,
  "timeout": "30s",
  "defaultSchema": "public"
}
```
