## ListPublicGrants

Lists the privileges available to PUBLIC within a Database — privileges every role implicitly holds regardless of its own grants. Returns the effective database-level PUBLIC privileges (CONNECT/TEMPORARY, synthesized from PostgreSQL defaults when datacl is unset) plus every explicit "GRANT ... TO PUBLIC" ACL entry on schemas and objects. Categorical defaults that are not per-object enumerable (EXECUTE on every function, the public-schema USAGE default) are not returned as rows; the UI surfaces those as guidance. Completes the "no direct grants does not mean no access" story.
